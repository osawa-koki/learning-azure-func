# learning-azure-func

Azure Functionsを学習する。  

## 手順 (自分用メモ)

`ms-azuretools.vscode-azurefunctions`拡張機能をインストール。  
その拡張機能のタブを開いてAzurenにログイン。  

WORKSPACEから新規関数を作成する。  
雲のデプロイマークからデプロイ可能。  

非常に簡単。
